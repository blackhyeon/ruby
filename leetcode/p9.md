# 9. Palindrome Number
* 문제 : https://leetcode.com/problems/palindrome-number/description/
```ruby
# @param {Integer} x
# @return {Boolean}
def is_palindrome(x)
    if x < 0
        return false
    end
    target = x.to_s
    len = target.length / 2
    for i in 0...len
       if target[i] != target[target.length - 1 -i]
           return false
       end
    end
    return true
end
```
